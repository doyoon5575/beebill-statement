# 꿀벌빵 거래명세서 생성기

모바일에서 거래처명, 품목, 소비자가, 수수료율, 공급가, 수량을 입력해 A4 거래명세서를 만들고 PDF/JPG로 저장하는 PWA입니다. 세금계산서 발급 기능은 포함하지 않고, 거래명세서와 납품기록 관리에 맞췄습니다.

## 실행

```powershell
node .\server.mjs
```

브라우저에서 `http://127.0.0.1:4173`을 엽니다.

별도 패키지 설치는 필요 없습니다.

## 포함 기능

- 새 거래명세서 작성
- 거래처 자동완성 및 저장
- 상품 자동완성 및 저장
- 사업자 정보 기본 저장
- 거래명세서별 공급자 정보 임시 수정
- 도장 이미지 업로드 및 출력물 자동삽입
- 데이터 백업/복원으로 PC와 휴대폰 간 기록 이동
- 품목 여러 개 추가
- 소비자가와 수수료율 기반 공급가 자동계산
- 공급가 직접입력 모드
- 거래처별 기본 수수료율
- 부가세 없음 / 포함 / 별도 계산
- A4 거래명세서 미리보기
- PDF 다운로드
- JPG 다운로드
- 발송상태 및 발송방법 기록
- 거래처명, 날짜, 금액 검색
- 월별 발행건수, 거래처 수, 총 금액 합계
- PWA 설치용 manifest 및 service worker

## 데이터 저장

현재 MVP는 브라우저 `localStorage`에 저장합니다. Supabase로 이전할 때 사용할 기본 테이블 SQL은 `supabase/schema.sql`에 있습니다.

같은 웹 주소라도 PC와 휴대폰은 브라우저 저장소가 서로 다릅니다. 기존 PC 기록을 휴대폰에서 보려면 앱의 `데이터` 화면에서 백업 파일을 저장한 뒤, 휴대폰에서 같은 화면의 백업 파일 가져오기로 복원합니다.

## 계산 예시

`꿀벌빵 12개입`, 소비자가 `29,000원`, 수수료율 `30%`, 수량 `12박스`를 입력하면 다음처럼 계산됩니다.

- 수수료 금액: `8,700원`
- 공급단가: `20,300원`
- 총 공급금액: `243,600원`

직접입력 모드에서는 사용자가 입력한 공급가가 우선 적용됩니다.
