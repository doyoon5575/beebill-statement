# 꿀벌빵 거래명세서 생성기

모바일에서 거래처명, 품목, 수량, 단가를 입력해 A4 거래명세서를 만들고 PDF/JPG로 저장하는 PWA입니다. 세금계산서 발급 기능은 포함하지 않고, 거래명세서와 납품기록 관리에 맞췄습니다.

## 실행

```powershell
node .\server.mjs
```

브라우저에서 `http://127.0.0.1:4173`을 엽니다.

별도 패키지 설치는 필요 없습니다.

## 모바일 웹 주소 만들기

GitHub 저장소를 Vercel에 연결하면 휴대폰에서 접속 가능한 `https://프로젝트명.vercel.app` 주소가 생성됩니다.

권장 Vercel 설정:

- Framework Preset: `Other`
- Build Command: 비움
- Output Directory: `.`
- Install Command: 비움

## 포함 기능

- 새 거래명세서 작성
- 거래처 자동완성 및 저장
- 상품 자동완성 및 저장
- 품목 여러 개 추가
- 부가세 없음 / 포함 / 별도 계산
- A4 거래명세서 미리보기
- PDF 다운로드
- JPG 다운로드
- 발송상태 및 발송방법 기록
- 거래처명, 날짜, 금액 검색
- 월별 발행건수, 거래처 수, 총 금액 합계
- PWA 설치용 manifest 및 service worker

## 데이터 저장

현재 MVP는 브라우저 `localStorage`에 저장합니다. Supabase로 이전할 때 사용할 기본 테이블 SQL은 `supabase/schema.sql`에 있습니다.

## 검증

2026-05-11 기준으로 다음 흐름을 확인했습니다.

- `node --check .\app.js`
- `node --check .\server.mjs`
- `http://127.0.0.1:4173` 응답 확인
- 모바일 폭 390px에서 본문 가로 밀림 없음
- 상품 자동선택: `꿀벌빵 | 5개입 | 12,000원`
- A4 미리보기 합계 `36,000원` 반영
- JPG 다운로드 생성
- PDF 다운로드 생성
- 기록 검색 입력 중 포커스 유지

## 세무 안내 문구

거래명세서 하단에는 다음 문구가 자동으로 들어갑니다.

> 본 거래명세서는 납품 및 거래내역 확인용이며, 세금계산서·현금영수증 등 세무 증빙은 별도 발급이 필요할 수 있습니다.
